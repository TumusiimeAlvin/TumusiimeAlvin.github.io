HTML Tags I've Learned

<header>
Defines introductory content or navigation links for a page or section.
<nav>
Defines a section of navigation links.
<section>
Defines a thematic grouping of content, typically with a heading.
<article>
Defines independent, self-contained content that could stand alone.
<footer>
Defines footer content for a page or section, such as author info or copyright.
